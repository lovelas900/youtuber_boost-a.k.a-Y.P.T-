🎬 Смотреть видео
📋 Как работают localhost и маска сети:
⚠️ Без прокси эти методы НЕ работают!
1. Нажмите «Запустить прокси» выше
2. Дождитесь статуса «Прокси запущен»
3. Выберите метод Localhost:8080
4. Вставьте ID видео и нажмите ▶

Localhost (127.0.0.1:8080) — ваш компьютер притворяется YouTube.
Прокси принимает запросы на 127.0.0.1 и перенаправляет их на реальный YouTube.

Маска сети (192.168.1.1:8080) — то же самое, но через адрес домашней сети.
Полезно если 127.0.0.1 блокируется.

# Пример как это работает: # Ваш браузер → 127.0.0.1:8080 → Прокси → YouTube.com # ↑ # (должен быть запущен!)
