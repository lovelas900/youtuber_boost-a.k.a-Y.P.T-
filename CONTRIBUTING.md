# 🤝 Как помочь проекту

Спасибо за желание помочь! Вот как это сделать.

## Добавление нового метода обхода

1. **Форкните репозиторий**
2. **Создайте ветку**: `feature/название-метода`
3. **Добавьте метод** в соответствующие файлы:
   - BAT файл (для Windows)
   - HTML файлы (для всех платформ)
4. **Проверьте** что метод работает
5. **Создайте Pull Request**

## Структура метода в BAT

```batch
:NEW_METHOD
cls
echo  ╔══════════════════════════════════════════════════════════════╗
echo  ║       Название метода                                       ║
echo  ╚══════════════════════════════════════════════════════════════╝
echo.
echo  Описание как работает метод...
echo.
echo  [1] Действие 1
echo  [2] Действие 2
echo.
set /p choice="  Выбор: "
:: ... код метода ...
goto MENU
