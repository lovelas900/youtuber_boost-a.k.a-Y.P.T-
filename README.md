## Что это?

**YouTube Proxy Toolkit** — это набор инструментов для обхода блокировки YouTube на любых устройствах. Не требует установки, работает через браузер, и предоставляет **более 20 различных методов** обхода блокировок.

### ✨ Особенности

- 🚀 **Без VPN** — большинство методов работают без VPN-сервисов
- 📦 **Нет зависимостей** — только HTML + CSS + JavaScript (и BAT для Windows)
- 🌍 **Кроссплатформенность** — Windows, Android, iOS, Linux
- 🔒 **Безопасность** — открытый исходный код, ничего не крадёт
- ⚡ **Быстро** — выбирайте самый быстрый метод из 20+
- 🆓 **Бесплатно** — MIT лицензия, никаких платных функций
- 📱 **PWA поддержка** — устанавливается как приложение на телефон
