🔐 VPN для Android
Установите одно из приложений:
🔗 Прокси Android
Настройка прокси на Android:
1. Настройки → Wi-Fi
2. Долгое нажатие на сеть → Изменить
3. Дополнительно → Прокси → Вручную
4. Введите IP и порт
🖥️ VPS SSH Туннель
Приложения для SSH туннеля:
• Termius (SSH клиент)
• ConnectBot (бесплатный)
• JuiceSSH (популярный)
ssh -D 1080 -N user@ваш_vps_ip

После подключения: прокси SOCKS5 127.0.0.1:1080

🛡️ ByeDPI для Android
ByeDPI — обходит DPI-блокировки провайдера. Для Android нужна версия с GitHub.
Инструкция:
1. Скачайте APK с GitHub Releases
2. Установите (разрешите неизвестные источники)
3. Запустите и нажмите "Start"
4. Прокси будет на 127.0.0.1:1080

После запуска ByeDPI настройте Wi-Fi прокси на 127.0.0.1:1080 (SOCKS5)
