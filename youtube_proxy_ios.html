🔗 Прокси на iOS
Как настроить прокси на iPhone/iPad:
1. Настройки → Wi-Fi
2. Нажмите (i) рядом с сетью
3. Прокси → Вручную
4. Введите сервер и порт
🖥️ VPS SSH Туннель
Приложения SSH для iOS:
• Termius (бесплатный)
• Shelly (SSH клиент)
• WebSSH
ssh -D 1080 -N user@ваш_vps_ip # Затем прокси SOCKS5: 127.0.0.1:1080
🔧 Смена DNS на iOS
Публичные DNS (могут обходить блокировки):

• Cloudflare: 1.1.1.1 / 1.0.0.1
• Google: 8.8.8.8 / 8.8.4.4
• AdGuard: 94.140.14.14
Как сменить DNS на iPhone/iPad:
1. Настройки → Wi-Fi
2. Нажмите (i) рядом с сетью
3. Прокрутите до DNS → Настроить DNS
4. Выберите Вручную
5. Удалите старые серверы
6. Добавьте: 1.1.1.1 и 1.0.0.1
